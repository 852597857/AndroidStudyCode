# AndroidStudyCode

记录一些 Android 相关的学习知识和原理，需要的童鞋可以看看。有bug请提Issue哇！

## 运行须知

运行本代码的童鞋，需要先以 Library 的方式导入 [AndroidUtils](https://github.com/D-clock/AndroidUtils) , 因为本库部分代码依赖于 AndroidUtils 。请知悉！
